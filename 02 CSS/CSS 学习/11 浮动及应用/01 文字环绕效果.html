纲手喜欢赌博，背后衣服上有一个“赌”字，不过赌博的运气都很差，被大家取了个外号叫做“传说中的大肥羊”。纲手是一个精通医疗忍术的女忍者，擅长体术和医疗忍术，拥有给予对方致命一击的“怪力”（将大量查克拉精确细致地提炼出来并迅速把其燃烧），在医疗忍术方面技术非常高超，堪称《火影忍者》里的“医圣”；因为讨厌自己年老的样子，所以用特别的医疗忍术来易容。有时还会随机应变装成10多岁、30多岁、40多岁的样子来逃避债主。她不单单只是优秀的医疗忍者，更有出色的战斗力，在第四次忍界大战中有着活跃表现。 她有时喜欢喝酒、睡觉、偷闲，故钢子铁及神月出云对纲手偷懒的行为感到不满。鸣人与自来也修行离开两年半，而她也在火影岩上刻上自已的雕像，将一身忍术及其技能传授给春野樱，随后在雨隐村的一次任务中，他们三人偶遇三名雨隐村的孤儿，分别是弥彦、小南和长门。原本大蛇丸想杀死三个孤儿，但他们因为有自来也的维护而存活了下来。 在战斗期间，斑得知纲手是初代的子孙并要把她先收拾掉。斑称纲手这程度的医疗忍术只不过苟延残喘一会罢了，和柱间比起来简直有着极大的差距，作为柱间的子孙没有什么可以拿出手，连木遁也不会，医疗忍术也不如柱间，甚至还是个弱女子。纲手回答：“我虽然是初代火影的子孙，但是我的确不会木遁，同初代不用结印的医疗忍术比起来我也没什么了不起的。但是我可不是一个弱女子，我拥有的不是纯粹的力量，而从初代那里传承下来的火之意志才是我真正的力量，别小看火之意志！”随后以怪力击败斑的木遁分身，却被斑以须佐能乎刺穿身体，但是她斩断须佐能乎后，“百豪之术”虽然使伤口自动恢复，却还面对着五个斑的木分身。最后五影被宇智波斑击败，纲手的身体也被宇智波斑用树界降延打成两段，重伤之下召唤蛞蝓为自己和其他四影疗伤。 [20]
