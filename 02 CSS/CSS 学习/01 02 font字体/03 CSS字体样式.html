成都尚医信息科技有限公司

企业介绍

成都尚医信息科技有限公司成立于2014年10月，是由华西医大外科毕业，美国哥伦比亚大学MBA）企业家携手在职资深外科医生共同创办，现今公司已有60余人。创始人曾在美国Charleston MedicalCenter 和Cleveland Clinic等机构工作，大量核心员工来自世界500强。

公司创始团队不但具备临床医学背景，欧美留学工作经历，哥大商学院全职MBA学位，而且拥有在中国成功创业的经验。IT、推广、医学是公司三大核心部门，其中医学部人员加入尚医之前全部在大型三甲医院临床一线工作，一半以上拥有临床内科或外科博士学位。IT、推广团队的大部分核心成员来自强生、罗氏、华为、赛门铁克等知名企业。

产品品牌

“术康医生”（Recovery Plus）是一款基于移动终端的跨平台“全流程患者管理工具”，为医生在患者教育、在院管理至术后康复指导、随访数据统计分析的全流程工作中提供专业协助，同时以数据为依据为医生/科室打造数字化品牌。

“术康”App是一款覆盖术前教育至术后康复全程专业指导的手机软件，帮助患者在主治医生指导下完成科学康复。患者通过“术康”App，从就医伊始即获得主治医生的关注，获得主治医生为其量身定制的康复计划，并可以及时联系主治医生，了解相关疾病知识、学习国外和其他患者的康复经验，通过“术康”的智能提醒积极管理自己的康复过程。
